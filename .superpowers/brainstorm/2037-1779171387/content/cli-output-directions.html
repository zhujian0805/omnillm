CLI output direction

Previewing three ways to make omnillm status, doctor, and list output cleaner and easier to scan.

A — Framed sections

Framed sections

Traditional CLI look with visible boxes around each section and stronger boundaries between summary blocks and tables.

Summary + clean tables

Top-level facts become a compact summary band, with boxed tables only where tabular data actually helps.
